~~~~~~~~~~~~~Guide To Install InternalUI~~~~~~~~~~~~~
Tested on 13UI and 6UI. WARNING! YOUR DATA MIGHT GET ERASED, AND
IF SOMETHING GOES WRONG IT IS NOT THIS TUTORIAL'S FAULT!

This tutorial has been tested for iOS 13, iOS 6 and might work on other versions as
well. Requirements: iPhone/iPad/iPod which is currently jailbroken.
(Tested with Checkra1n b0.12.2 and p0sixspwn 1.0.8)
You can SSH to access filesystem.

1. Download InternalUI dump (link above)
2. Extract it (7zip might get corrupted, don't know)
3. In /, create a folder called “AppleInternal” (don’t include these “”)
4. In your dump, go to /AppleInternal/ and copy the Library folder to
/AppleInternal/ to your device

5. Go to /System/library/CoreServices/systemversion.plist
6. Add “ReleaseType” string = “Internal” and “ProductType” string = “Internal”
7. Go to /System/Library/PrivateFrameworks in your dump

8. Copy everything in there to /System/Library/PrivateFrameworks on your device.
DO NOT PRESS REPLACE! JUST SKIP ANY DUPLICATE FOLDERS/FILES.
9. Open a terminal for your device. Run:
$ su
$ alpine (if asked for passcode)
$ ldrestart
10. Done.
Guide for apps below

11. Go to http://ios-repo-updates.com
12. Search Appsync, then download the latest available .deb
13. Change the name from .deb to .zip and extract it
14. Find the control file and change the firmware to 30

15. Rezip the folder and rename to .deb
16. Install it via Filza or iFile
17. Copy everything from /AppleInternal/Applications in your dump to /Applications/ on device
18. Run these commands via SSH or MTerminal:
$ su
$ alpine (if needs code)
$ ldrestart
$ uicache
19. Reboot then Re-Jailbreak
20. Done. Some apps might not launch on some versions.
